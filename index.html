Attack Momentum

  • SofaScore doesn't provide graphs for all matches.
  • Scoreboards/Stats are updated automatically.
  • Use zoom for better viewing (CTRL + ScrollDown/ScrollUp).
  • Drag/Select a match card by clicking/holding at the background of it.
  • Other Version (with more stats, and more bugs)

Attack Momentum is a project that uses SofaScore API to provide live football match scoreboards and stats.

This version was entirely made with vanilla HTML/JS/CSS, so the code may be a mess...
